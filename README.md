# Grocery Booking API

This is a **Spring Boot-based Grocery Booking API** that provides CRUD operations for grocery items, user management, and order processing. It includes a **Docker setup** for easy deployment.

## Tech Stack
- **Backend:** Java 17, Spring Boot (v3.4.3)
- **Database:** MySQL  
- **ORM:** Hibernate  
- **Build Tool:** Maven  
- **Containerization:** Docker  

## Setup & Run Locally

### 1. Clone the Repository and Build the Project

### 2. Run the docker command 
```bash
docker-compose up -d
```

This will:

-Start the application

-Start a MySQL database
